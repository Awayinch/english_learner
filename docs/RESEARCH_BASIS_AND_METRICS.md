# 研究依据与指标映射

## 1. 语言学习与词汇知识

- Nation 的词汇知识框架强调词汇学习不仅是认识词义，还包括 form、meaning、use。本项目据此将卡片完整率定义为词形、释义、词性与语境证据的组合完整度。
- Nation 的 Four Strands 强调输入、输出、语言聚焦学习和流利度发展之间的平衡。本项目将语境暴露视为输入痕迹，将主动使用视为输出痕迹。
- Laufer 与 Hulstijn 的 Involvement Load Hypothesis 强调词汇保持与任务投入有关。后续语境自动化增强可以围绕 need、search、evaluation 补充同义词、词根和例句。

## 2. 记忆与复习心理学

- Retrieval practice 说明主动回忆比单纯重读更能促进长期保持。因此“主动回忆成功率”必须来自测验日志，不能从词汇卡片数量直接推断。
- Spacing effect 说明分散复习优于集中复习。因此“间隔复习逾期数”需要记录 nextReviewAt、lastReviewedAt 和 reviewCount。
- Cognitive Load Theory 提醒看板不能过度堆叠信息。本项目将看板拆为已观测指标、待采集指标和 DIKW 流转，以降低解释负担。
- 认知负荷不应由词汇数量推断，而应通过学习者自评或任务表现采集。本项目用 1-5 分滑杆记录学习后的主观负荷，作为轻量过程数据。

## 3. 自我调节学习与行为记录

- Self-Regulated Learning 强调计划、监控和反思。系统中的学习会话、词卡、导出记录可作为学习者监控自身知识资产的依据。
- 新增学习反思/自评记录模块用于采集计划、监控和反思证据：学习者记录认知负荷、学习焦虑、复习信心和下一步计划。
- Learning Analytics 强调对学习者及其情境数据进行测量、收集、分析和报告。因此看板只展示可由本地日志支持的指标，并显式标注缺测指标。

## 4. 指标操作化

| 指标 | 当前状态 | 计算方式 | 理论关联 |
| --- | --- | --- | --- |
| 词汇卡片 | 已观测 | 本地 vocabulary 数量 | 知识外化 |
| 语境暴露 | 已观测 | 目标词在历史消息中出现次数 | meaning-focused input |
| 主动使用 | 已观测 | 目标词在用户消息中出现次数 | meaning-focused output |
| 卡片完整率 | 已观测 | 完整词卡 / 总词卡 | form-meaning-use |
| 待强化节点 | 已观测 | 待查询词 + 不完整词卡 | self-monitoring |
| 认知负荷 | 已采集 | 学习者 1-5 分自评均值 | cognitive load / SRL |
| 学习焦虑 | 已采集 | 学习者 1-5 分自评均值 | affective factors / SRL |
| 复习信心 | 已采集 | 学习者 1-5 分自评均值 | self-efficacy / SRL |
| 主动回忆成功率 | 待采集 | 正确回忆次数 / 测验次数 | retrieval practice |
| 间隔复习逾期数 | 待采集 | nextReviewAt 早于当前日期的卡片数 | spacing effect |

## 5. 报告写法提示

看板中的数字不应表述为“已经掌握多少词”，而应表述为“系统当前观测到的知识资产、学习行为痕迹和学习者自评状态”。真正的掌握度需要结合测验、复习间隔和长期保持数据。

## 6. 语境自动化增强模块

语境自动化增强模块用于体现知识管理中的“组合化”和“深加工”：

- 将单一词条与同义表达、词根/构词提示、词族候选、例句和来源语境组合为完整知识卡片。
- 对历史对话进行目标词匹配，优先保留真实学习语境。
- 对缺少真实语境的词条生成 IELTS 场景例句，并标注增强方法，避免将其误写为真实使用证据。
- 后续可接入外部词典 API 或模型调用，将本地规则增强升级为可溯源词典增强。
